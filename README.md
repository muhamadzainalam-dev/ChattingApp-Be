# ChattingApp – Backend 

[![Node.js](https://img.shields.io/badge/Node.js-Backend-339933?style=for-the-badge&logo=node.js&logoColor=white)](https://nodejs.org/)
[![Express](https://img.shields.io/badge/Express-4-black?style=for-the-badge&logo=express&logoColor=white)](https://expressjs.com/)

Node.js backend for realtime chat using **Socket.IO**, designed to support message events and client synchronization.

> 🔗 **Frontend Repo:** `muhamadzainalam-dev/ChattingApp-Fe`  
> `https://github.com/muhamadzainalam-dev/ChattingApp-Fe`

---

## 💼 Skills & Tech Used

- **Server:** Node.js
- **API Framework:** Express
- **Realtime:** Socket.IO
- **Middleware:** CORS

> Note: `socket.io-client` is also listed in this backend repo’s `package.json`.

---

## 🚀 Run Locally

```bash
git clone https://github.com/muhamadzainalam-dev/ChattingApp-Be.git
cd ChattingApp-Be
npm install
node index.js
```
